# Gestor Comercial

Base inicial de um sistema SaaS comercial em `Next.js`, `TypeScript` e `Tailwind CSS`, com estrutura modular e layout persistente.

## Estrutura inicial

- `src/app`: rotas e layouts do App Router
- `src/components`: componentes compartilhados de layout
- `src/config`: configurações centralizadas
- `src/modules`: organização modular por domínio

## Módulos

- `estoque`
- `cadastros`
- `vendas`
- `financeiro`
- `usuarios`
- `logistica`
- `configuracoes`
